# Code of Conduct

Rally is a small open source project.
Keep the work direct, respectful, and useful.

## Our standard

When you take part in issues, pull requests, and discussions here:

- be respectful
- stay on the technical topic
- assume good intent until the facts say otherwise
- give clear feedback without personal attacks
- leave room for people who are still learning

## Not acceptable here

These are not okay:

- harassment, threats, or hate speech
- insults, name-calling, or pile-ons
- sexual content or unwanted sexual attention
- posting private information without permission
- trolling, spam, or repeated bad-faith arguments
- sharing exploit details in public when the report should be private

## Scope

This covers Rally project spaces, including GitHub issues, pull requests,
discussions, and review threads.

## Enforcement

The maintainer may edit, hide, lock, or remove content that breaks this code of
conduct.
The maintainer may also block repeat offenders from project spaces.

If you see a conduct problem, flag it on GitHub when you can.
If the problem needs a private path, ask for moderator help without reposting
the harmful details in public.
